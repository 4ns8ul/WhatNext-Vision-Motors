
# WhatsNext Vision Motors – Salesforce CRM Implementation 🚗⚙️

A Salesforce-based CRM system built for WhatsNext Vision Motors to automate vehicle order processing, dealer assignment, test drive management, and service tracking. Developed during a virtual internship project to modernize operations and enhance customer experience.

---

## 📌 Project Overview

This project focuses on digitizing core operations of WhatsNext Vision Motors using Salesforce. Key modules implemented include:

- Vehicle Order Management
- Dealer Auto-Assignment Flow
- Test Drive Reminder Flow
- Service Request Handling
- Inventory & Customer Tab Setup

All workflows, automations, and triggers are implemented using Apex, Process Builders, and Salesforce Flows.

---

## 🚀 Features

- **Automated Dealer Assignment** on vehicle order creation
- **Batch Apex** for efficient processing of order records
- **Scheduled Apex Jobs** for periodic automation
- **Test Drive Reminders** via flows
- **Validation Rules** and **Custom Objects** for Vehicles, Orders, Dealers
- **Trigger Handler Framework** for clean Apex logic

## ⚙️ Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/4ns8ul/WhatNext-Vision-Motors.git
   ```
2. Open in Salesforce Developer Console.
3. Create required custom objects:
   - Vehicle
   - Vehicle Order
   - Dealer
   - Test Drive
4. Add Apex Classes & Triggers from `/Source Code`
5. Import Flows manually via Flow Builder using screenshots as reference.
